# CLAWD sprite attribution

The four GIF sprites in this directory (`clawd-idle.gif`,
`clawd-headphones-groove.gif`, `clawd-react-double-jump.gif`,
`clawd-happy.gif`) are pixel-art renditions of **CLAWD**, the Claude Code
companion mascot.

## Character
**Clawd** is a character belonging to **Anthropic, PBC**
(https://www.anthropic.com). Used here in a non-commercial personal blog
context (fan-art, decorative companion, no implied affiliation or endorsement).

## Sprite source
Fan-created pixel art extracted from the **Clawd on Desk** project by
**鹿鹿 (rullerzhou-afk)**:
https://github.com/rullerzhou-afk/clawd-on-desk

GIFs in `assets/gif/` of that repository, cropped to 170×170 to remove
transparent canvas padding. No modification to the artwork itself.
